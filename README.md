# Colorization of GrayScale Images using GAN

A deep learning project that automatically colorizes grayscale images using a **Generative Adversarial Network (GAN)**. The model learns the mapping from grayscale images to realistic RGB images, producing visually plausible colorized outputs while preserving image structure and details.

**GitHub Repository:**
https://github.com/Pravin1105/Colorization-of-GrayScale-Images-using-GAN

---

## Overview

Image colorization is the task of converting a grayscale image into a colored image by predicting the missing chrominance information while preserving the luminance information.

This project implements a **Generative Adversarial Network (GAN)**, where:

* **Generator:** Learns to generate realistic color images from grayscale inputs.
* **Discriminator:** Learns to distinguish between real color images and generated colorized images.

Both networks compete during training, allowing the generator to produce increasingly realistic colorizations.

## Dataset

The model is trained using images from the **Places365 Dataset**.

### About Places365

Places365 is a large-scale scene recognition dataset developed for visual scene understanding. It contains millions of images covering a wide variety of indoor and outdoor environments, making it well suited for learning realistic image colorization.

The dataset provides diverse scenes such as:

* Streets
* Mountains
* Forests
* Buildings
* Beaches
* Rooms
* Natural landscapes
* Urban environments

This diversity enables the GAN to generalize well across different image categories.

### Dataset Source

**Kaggle:**
https://www.kaggle.com/datasets/benjaminkz/places365
